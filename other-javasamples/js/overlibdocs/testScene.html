overlib popups:  
Pop Me This Frame

Pop Me Cross Frame

Plugin module overlibmws_iframe.js is being used to prevent obscuring of overlib popups for IE v5.5 or higher via an iframe shim (as opposed to iframe content).  For other old versions of browsers that need it, plugin module overlibmws_hide.js also is imported and its HIDESELECTBOXES and HIDEBYIDNS4 commands are used to hide and restore the form element if needed in conjunction with display and removal of the popup in the cross frame, whereas its HIDEBYIDALL command is used to hide the system control (flash object) in this frame.  Also see examples with nested framesets, as well as an example which deals with the flash object by setting its wmode to transparent.

(for IE 5.5 or higher only)
 window.createPopup:
Create Me This Frame

Create Me Cross Frame

The window.createPopup method was developed by Microsoft for creating popup windows with DHTML functionality but not subject to being obscured by any form elements or by objects with system controls.  The method is restricted to IE v5.5 or higher.  The overlibmws_iframe.js plugin module is equally effective for overlib popups, but has the same browser and version restriction.
